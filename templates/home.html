Conjunto de datos de luces de tránsito

Este conjunto de datos está compuesto de 3000 imágenes, 2600 de ellas son de entrenamiento, y 400 de prueba. El objetivo de este conjunto de datos es identificar la ubicación y el tipo de luces de tránsito que aparecen en una imagen.

Esta herramienta de visualización permite ver una tabla con todas las imágenes y sus características. También permite ver una imagen en particular, con un rectángulo resaltando la posición de las luces de tránsito detectadas.
